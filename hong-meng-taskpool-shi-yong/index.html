鸿蒙TaskPool使用

2024-08-04

注意事项

1、在ets文件中使用**@Concurrent进行标注的函数任务才能放到TaskPool中使用
2、需要在taskpool中执行的自定义类方法需要使用@Sendable进行标注，仅支持在ets文件中使用
3、在taskpool中执行的方法耗时不能超过3分钟**
4、入参仅支持满足序列化的类型，详细列表：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/serialization-support-types-V5#TaskPool%E5%92%8CWorker%E6%94%AF%E6%8C%81%E7%9A%84%E5%BA%8F%E5%88%97%E5%8C%96%E7%B1%BB%E5%9E%8B
5、不同线程上下文不同，因此taskpool只能使用线程安全的库，涉及UI相关非线程安全库不能使用
6、传输数据量限制大小为16M

代码示例

taskpool的用法比较简单，常用的就是execute方法，在子线程中执行任务，结果返回给主线程，需要注意前面注意事项

@Concurrent
function imageProcessing(dataSlice: ArrayBuffer): ArrayBuffer {
  // 步骤1: 具体的图像处理操作及其他耗时操作
  console.log('imageProcessing');
  return dataSlice;
}

function histogramStatistic(pixelBuffer: ArrayBuffer): void {
  console.log(`histogramStatistic`);
  // 步骤2: 分成三段并发调度
  let number: number = pixelBuffer.byteLength / 3;
  let buffer1: ArrayBuffer = pixelBuffer.slice(0, number);
  let buffer2: ArrayBuffer = pixelBuffer.slice(number, number * 2);
  let buffer3: ArrayBuffer = pixelBuffer.slice(number * 2);

  let group: taskpool.TaskGroup = new taskpool.TaskGroup();
  group.addTask(imageProcessing, buffer1);
  group.addTask(imageProcessing, buffer2);
  group.addTask(imageProcessing, buffer3);

  taskpool.execute(group, taskpool.Priority.HIGH).then((ret: Object) => {
    // 步骤3: 结果数组汇总处理
    console.log(`taskpoll execute result: ${JSON.stringify(ret)}`);
  })
}

@Entry
@Component
struct Index {
  aboutToAppear(): void {
    console.log('aboutToAppear')
    let buffer: ArrayBuffer = new ArrayBuffer(24);
    histogramStatistic(buffer);
  }
}

接口文档

(https://developer.huawei.com/consumer/cn/doc/harmonyos-references-V5/js-apis-taskpool-V5)[https://developer.huawei.com/consumer/cn/doc/harmonyos-references-V5/js-apis-taskpool-V5]

除了第二节讲到execute方法之外，接口文档中还提供以下方法，这些方法在实际开发也会使用：

  • taskpool.executeDelayed： 延迟执行
  • taskpool.executePeriodically：周期任务执行

taskpool的使用最重要是要进行抽象，将可分离的计算任务抽出来，交给线程池执行，不阻塞主线UI线程加载和绘制，实现并发加载数据，不同于异步加载数据。
