2024-08-04

HiDumper命令

查看界面的节点树

hdc shell param set persist.ace.debug.enabled 1 # 开启ArkUI的debug模式
hdc shell hidumper -s WindowManagerService -a '-a' # 查看对应的winId
hdc shell hidumper -s WindowManagerService -a '-w [windId] -element -c'
hdc shell find /data/ -name arkui.dump
// 以下路径为find命令查找的路径
hdc file recv /data/app/el2/100/base/com.example.demo/haps/entry/files/arkui.dump . // 获取文件到本地

查看内存信息

hdc shell hidumper -s WindowManagerService -a '-a' 获取到当前应用的pid
hdc shell hidumper --mem [pid]

和安卓应用内存信息，开发一般情况下只需要关注PSS （Proportional Set Size，实际使用物理内存）Total一列的数据，即示例应用实际使用的物理内存。

查看CPU信息

hdc shell hidumper -s WindowManagerService -a '-a' 获取应用的pid
hdc shell hidumper --cpuusage [pid]

一般情况下，只需要关注

  • Total Usage（总使用率）: 用于统计应用的总CPU使用率

  • User Space（用户程序的运行空间）: 用于执行简单的运算

  • Kernel Space（内核的运行空间）: 用于调用系统的资源

    如果想要查看CPU在一段时间内的使用率，则可以通过Shell脚本，多次执行 hdc shell hidumper --cpuusage [pid] 命令，并通过 hdc shell hidumper --zip --cpuusage 将结果输出到手机的/data/log/hidumper目录下，方便进行分析。
