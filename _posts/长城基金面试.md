一道数据库的题目：详细可以看[这篇文章](<https://blog.csdn.net/waiwai021/article/details/50171669>)

**表中有A B C三列,用SQL语句实现：当A列大于B列时选择A列否则选择B列，当B列大于C列时选择B列否则选择C列。**

这道题主要用到`case`函数

```sql
select case where a>b then a
			where b>c then b
		else c
from table;
```





另外一个数据库的知识点记录：

关于分组的使用，GROUP BY把具有相同的数据值的行放到一起，HAVING对分组进行过滤，WHERE则是对行进行过滤

```sql
SELECT col, COUNT(*) AS num
FROM mytable
WHERE col > 2
GROUP BY col
HAVING num >= 2;
```

COUNT(\*) 是统计相同分组中的个数，GROUP BY一般用在WHERE自语句
